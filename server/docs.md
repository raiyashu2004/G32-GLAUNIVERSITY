*** Inventory Management System ***

Introduction: This thing provides a comprehensive overview of what about repository is all about. It creates a Inventory Management System built using MERN Stack. It covers different aspects of the project i.e Frontend Backend and Database.

Tech Stacks Used: 

* Front end
    React
    Vite

* Back end
    Node
    Express
    MongoDb
    Mongoose


Starting the Server:

  1. Backend:

       cd server
       npm install

  2. Frontend:

       cd Inventory_Management_System_1
       npm install

Running the Server:

  1. Backend:

      cd server
      npm run dev

  2. Frontend:

     cd Inventory_Management_System_1
     npm run dev
